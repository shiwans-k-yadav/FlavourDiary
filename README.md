# 🍲 RecipeTalk

A simple recipe search web app that fetches and displays recipes using TheMealDB API.

## 🚀 Features
- Search recipes by name
- View recipe image, category, and origin
- Fast and clean UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- TheMealDB API

## ▶️ Run Locally
Just open `index.html` in your browser.
